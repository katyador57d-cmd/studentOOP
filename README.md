# Менеджер студентов на PHP

CLI-приложение для управления списком студентов и их оценками.
Написано на PHP 8.1 в ООП-стиле.

## Возможности
- Добавление студентов
- Выставление оценок по предметам
- Поиск студента по имени
- Фильтрация и сортировка списка
- Отчёты по предметам и лучшим студентам
